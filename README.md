# Netflix Dataset Analysis

In this project, I analyzed a Netflix dataset. I mainly followed the guidelines below but also added some extra analysis for better insights.

---

## Tasks and Questions

### Data Cleaning Tasks
1. **Check for duplicate records**  
   - If duplicates exist, remove them.
2. **Check for null values**  
   - Visualize missing data with a heatmap.

---

### Analysis Questions

1. **For 'House of Cards'**  
   - Find the Show Id and the Director of this show.

2. **Yearly Releases**  
   - Determine in which year the highest number of TV Shows and Movies were released.  
   - Visualize with a bar graph.

3. **Count of Movies and TV Shows**  
   - How many Movies and TV Shows are in the dataset?  
   - Visualize with a bar graph.

4. **Movies Released in 2000**  
   - Show all Movies released in the year 2000.

5. **TV Shows Released in India**  
   - Show only the titles of all TV Shows released in India.

6. **Top Directors**  
   - Show the top 10 directors who contributed the highest number of TV Shows and Movies to Netflix.

7. **Filtered Records**  
   - Show all records where:
     - Category is Movie and Type is Comedies, or
     - Country is United Kingdom

8. **Tom Cruise Appearances**  
   - Count how many movies/shows Tom Cruise was cast in.

9. **Netflix Ratings**  
   - List all the different ratings defined by Netflix.

9.1 **TV-14 Movies in Canada**  
   - Count how many Movies got the 'TV-14' rating in Canada.

9.2 **R-Rated TV Shows after 2018**  
   - Count how many TV Shows got the 'R' rating after the year 2018.

10. **Maximum Duration**  
    - Find the maximum duration of a Movie/Show on Netflix.

11. **Top Country by TV Shows**  
    - Determine which individual country has the highest number of TV Shows.  
    - *Note:* Exploded the data because some records contain multiple countries in one cell.  
    - Visualized with a bar chart.

12. **Sort Dataset by Year**  
    - Show how the dataset can be sorted by the release year.

13. **Advanced Filtering**  
    - Find all instances where:
      - Category is 'Movie' and Type is 'Dramas', or  
      - Category is 'TV Show' and Type is 'Kids' TV'

---

## Additional Notes
- Exploded the 'Country' column to accurately count individual countries for TV Shows.  
- Added bar charts for visual analysis wherever applicable.  

---

This analysis provides insights on Netflix content distribution, ratings, top contributors, and popular countries for TV Shows.
